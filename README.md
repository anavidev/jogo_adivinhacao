# Jogo de Adivinhação

Jogo simples onde o jogador deve adivinhar um número sorteado pelo computador. O objetivo é acertar o número correto com o menor número de tentativas possível. O jogo oferece dicas se o número escolhido pelo jogador é maior ou menor que o sorteado.

## Funcionalidades

- **Sorteio de Números:** O computador sorteia um número de 0 a 100.
- **Seleção de dificuldade:** O jogador pode escolher o nível de dificuldade, que determina o número de tentativas disponíveis.
- **Dicas:** O jogo oferece dicas informando se o número que o jogador escolheu é maior ou menor que o número sorteado.
- **Mensagem Final:** Ao acertar, o jogo informa o número de tentativas necessárias. Caso contrário, revela qual era o número correto.

## Conceitos de programação

- **Estruturas de controle:** Uso de loops e condicionais para verificar as tentativas do jogador.
- **Manipulação de variáveis:** Armazenamento das tentativas e comparação dos valores inseridos.
- **Funções:** Organização do código em funções para garantir modularidade e facilitar a manutenção.

## Tecnologias Utilizadas

- **Python:** Linguagem de programação utilizada para desenvolver o algoritmo do jogo.
- **Biblioteca 'Random':** Biblioteca nativa do Python, usada para realizar o sorteio dos números de forma aleatória.

## Como Jogar

1. O computador sorteia um número de 0 a 100.
2. O jogador deve tentar adivinhar o número.
3. A cada tentativa, o jogo oferece uma dica, indicando se o número inserido é maior ou menor que o sorteado.
4. Se o jogador acertar dentro do número de tentativas permitidas, o jogo exibe uma mensagem informando o número de tentativas usadas.
5. Se o jogador não acertar dentro das tentativas permitidas, o jogo revela qual era o número correto.
